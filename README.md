# stuff for agents

## Skills

### What are skills?

Via [Cursor docs](https://cursor.com/docs/context/skills#what-are-skills):
> A skill is a portable, version-controlled package that teaches agents how to perform domain-specific tasks. Skills can include scripts, templates, and references that agents may act on using their tools.

### How to use skills

Depending on agent of choice:

#### Project level

`.{cursor,claude,github,agents}/skills`

#### User-level (global)

`~/.{cursor,claude,github,agents}/skills`
